# RosUltrasonic11V10
Sinusoidal ultrasonic node for ROS Melodic

Usage:

Install:
clone that git repo and create catkin workspace

1-mkdir catkin_ws1
2-cd catkin_ws1
3-mkdir src
4-cd src
5-catkin_init_workspace
6-cd ..
7-catkin_make
8-source devel/setup.bash

Run:
roslaunch ultrasonics ultrasonics.launch
